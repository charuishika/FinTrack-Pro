# FinTrack Pro

A full-stack personal finance dashboard that helps users track income, expenses, budgets, and financial insights.

## Features
- User Authentication (JWT)
- Income & Expense Tracking
- Budget Management
- Financial Dashboard
- Interactive Charts
- AI-Powered Insights

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- Recharts
- Gemini API

## Installation

### Backend
npm install
npm run dev

### Frontend
npm install
npm start
